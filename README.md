# Git-Automation
Lightweight Bash-based Git automation. Keeps your repositories up to date with minimal effort.

## Quick Start

1. **Download The Repo**

   ```bash
   cd /home/$USER/src
   git clone https://github.com/Tyonnchie-Berry-1996/Git-Automation.git
   ```
2. **Run It**
   ```bash
   cd Git-Automation/
   ```
3. **Test out some repo automation**
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```
